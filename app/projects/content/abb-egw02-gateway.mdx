## Overview

Contributed to the development of ABB's EGW-02 Connectivity Edge Gateway, a **Red Dot Award-winning** product that enables secure cloud connectivity for industrial drives and motors.

## My Contributions

### Secure Cloud Connectivity
- Architected secure cloud connectivity using TPM (Trusted Platform Module) and certificate-based authentication
- Implemented Azure IoT Hub integration with device provisioning, digital twin synchronization, and secure file upload
- Enabled predictive maintenance capabilities, **reducing downtime by 30%**

### Firmware Development
- Engineered firmware modules on LmP OS (Yocto-based) for real-time data exchange
- Developed OPC UA server/client modules for standardized industrial equipment data exchange
- Created multithreaded data aggregation pipelines for telemetry and events

### System Architecture
- Designed modular architecture with MQTT/ZeroMQ message-based IPC
- Optimized data pipelines achieving **40% throughput improvement** and **99.9% uptime**
- Established test frameworks with Google Test achieving **80%+ code coverage**

## Technologies Used

- **Languages**: C++17, C, Python
- **Protocols**: OPC UA, MQTT, ZeroMQ, Modbus
- **Cloud**: Azure IoT Hub, Device Provisioning Service
- **Security**: TPM 2.0, X.509 Certificates, TLS/SSL
- **OS**: LmP OS (Yocto-based Embedded Linux)
- **Tools**: CMake, Google Test, GDB, UART/I2C debugging

The ABB EGW-02 Connectivity Edge Gateway was awarded the prestigious **Red Dot Design Award** for its innovative industrial IoT capabilities and user-centric design.
