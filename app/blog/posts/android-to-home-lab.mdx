# üì± Turn Your Old Phone into a 24/7 Home Server

Here is the quick guide to converting your Android device into a Linux server using Termux.

### 1Ô∏è‚É£ Prepare the Phone

Open **Termux** and run these commands one by one:

```bash
# Update everything
pkg update && pkg upgrade -y

# Install SSH
pkg install openssh

# Set a password (you need this to log in!)
passwd

# Find your local IP (look for 192.168.x.x)
ifconfig

# Start the SSH Daemon (CRITICAL STEP)
sshd

```

### 2Ô∏è‚É£ Connect from Laptop

Open your terminal (PowerShell or Terminal) and type:

```bash
# Replace with your phone's IP
ssh -p 8022 192.168.1.XX

```

*Enter the password you set in Step 1.*

### 3Ô∏è‚É£ Transfer Your Code

You can either `git clone` your repo or upload a zip file.
**To upload a zip from your laptop:**

```bash
# Run this on your laptop, NOT inside the SSH session
scp -P 8022 my-app.zip 192.168.1.XX:~

```

**Back on the phone:**

```bash
pkg install unzip
unzip my-app.zip
cd my-app

```

### 4Ô∏è‚É£ Install Node.js & Run App

```bash
# Install Node.js (LTS version is best)
pkg install nodejs-lts

# Install dependencies
npm install

# Install PM2 (Process Manager)
npm install -g pm2

# Run your app (Adjust for your specific script, e.g., "start" or "dev")
pm2 start npm --name "my-app" -- start

```

### 5Ô∏è‚É£ Make it Public (Cloudflare Tunnel)

Expose your localhost to the world for free.

```bash
pkg install cloudflared

# This gives you a random public URL
cloudflared tunnel --url http://localhost:3000

```

### üí° Pro Tips

**üö´ Prevent the App from Sleeping**
Android kills background apps to save battery. To stop this:

1. Pull down your notification bar.
2. Tap "Acquire Wakelock" on the Termux notification.
3. *Or run this command:* `termux-wake-lock`

**ü™ü Multitasking (Tmux)**

